Syarat Dan Ketentuan Kereta Api Indonesia

1. Admin dan operator memiliki id dan paswordnya masing-masing

2. Admin dan operator dapat melihat mengubah data pemesanan masing-masing.

3. Admin dan operator dapat melihat data konsumen yang telah melakukan pemesanan.

4. Admin dapat melihat dan mencetak laporan hasil yang didapat dari pemesanan tiket kereta api.

5. Tersedia fasilitas menu searching atau pencarian data dari setiap menu yang disediakan guna memudahkan pekerjaan admin dan operator.

6. Admin dan operator silahkan melihat jadwal keberangkatan dan harga tiket kereta, sebelum memesan tiket.

7. Admin dan operator mengisi harga tiket yang sesuai ada di jadwal tiket
